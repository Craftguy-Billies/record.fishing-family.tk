Free Website Domain Setup Procedure
  1. Register a custom domain name at www.freenom.com or dot tk.
  2. Enter DNS records shown as above.
  3. Register/login Github account.
  4. Create new repository.
  5. Upload an "index.html" file to the new github repository.
  6. At settings→pages, choose "main" branch, then at custom domains type your registered domain name.
  7. A CNAME file should appear in your github code afterwards.
  8. Wait few minutes. Everything is set.
